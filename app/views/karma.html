Karma

  • Karma permite ejecutar código JavaScript en multitud de navegadores reales.
  • Resulta muy útil cuando se desea testear código en distintos navegadores.
  • Permite ejecutar tests localmente durante el desarrollo y en un servidor de integración continua.
  • Karma no es un framework para testing, ni una librería.
  • Karma lanza un servidor HTTP y genera un archivo HTML para la ejecución del test.
  • Se usa en conjunto con un framework de testing.
  • Karma da soporte a gran cantidad de navegadores (Chrome, Firefox, Safari, PhantomJS, IE, Opera...)
