Aplicación Single-Page

  • Una Single-Page Application (SPA) es una aplicación web que se ejecuta en una sola página, logrando una experiencia de usuario similar a una aplicación de escritorio.
  • Una sola carga de la página. La página no se recarga.
  • Los recursos se cargan dinámicamente en respuesta a acciones del usuario.
  • AngularJS ofrece los componentes necesarios para gestionar la navegación del usuario.
  • Con AngularJS puede asociarse una vista y un controlador a una URL.
