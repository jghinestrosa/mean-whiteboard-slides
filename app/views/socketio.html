Socket.IO

  • Socket.IO es un framework para Node.js pensado para aplicaciones en tiempo real.
  • Funciona en todas las plataformas, navegadores y dispositivos, centrándose tanto en la seguridad como en la velocidad.
  • Cuenta con una módulo para el servidor que se instala mediante npm, y de una librería para el lado del cliente.
  • Es uno de los frameworks más utilizados a la hora de trabajar con Node.js y WebSockets.
  • Da soporte cuando hay algún problema con los WebSockets, ya sea que el navegador no los implemente o que las conexiones sean bloqueadas por un antivirus o un firewall.
  • Puede suplir la falta de WebSockets haciendo uso de AJAX.
  • Ofrece una API con métodos que permiten realizar transmisiones broadcast, así como el uso de namespaces y salas.
