Grunt

  • Grunt es un gestor de tareas para JavaScript.
  • Grunt proporciona automatización.
  • Automatiza tareas repetitivas como minificación, compilación, ejecución de tests unitarios, revisiones de sintaxis.
  • Hace uso del archivo de configuración Gruntfile.jsen el que se registran las tareas y posteriormente pueden ejecutarse al ser solicitadas por el desarrollador.
  • Ofrece cientos de plugins, lo que permite automatizar cualquier tarea con un mínimo de esfuerzo
