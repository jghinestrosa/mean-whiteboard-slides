Express.js - Características

  • Middleware
    • Función JavaScript que gestiona las peticiones HTTP que recibe una aplicación de Express.
    • Node.js ofrece una única función para gestionar las peticiones.
    • Express permite tener varias funciones separadas y cada una de ellas puede encargarse de una parte del trabajo.
      • Peticiones de logging.
      • Enviar archivos estáticos.
      • Enviar cabeceras HTTP.
    • La funcionalidad se mantiene separada y la aplicación se vuelve más modular.
  • Routing
    • Express permite varias funciones para gestionar las distintas peticiones.
    • A diferencia del middleware, estas funciones se ejecutan condicionalmente segun la URL o método HTTP.
      • Una función para gestionar peticiones GET.
      • Una función para gestionar peticiones POST.
