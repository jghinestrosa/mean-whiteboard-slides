Yeoman

  • Yeoman es una herramienta pensada para facilitar el desarrollo de proyectos.
  • Se encarga de proporcionar al desarrollador una estructura inicial sobre la que basarse y comenzar el desarrollo.
  • Intenta solucionar la mayoría de problemas o la lentitud que conlleva configurar a mano un proyecto desde el principio.
  • Un generador de Yeoman es básicamente un plugin que permite crear el esqueleto base de un proyecto.
  • Para MEAN Whiteboard se ha utilizado el plugin angular-fullstack.
  • Existen generadores para distintos proyectos: Backbone.js, Express.js, Node.js, Polymer, Bootstrap, jQuery…
