Jasmine

  • Jasminees un framework para desarrollo guiado por comportamiento (Behaviour-Driven Development).
  • Permite escribir tests que denotan como debería comportarse el código
  • AngularJS proporciona algunos mockups, así como funciones de testing, para permitir al desarrollador crear servicios, controladores y filtros directamente en los tests unitarios.
  • AngularJS también permite simular objetos como HttpRequests.
