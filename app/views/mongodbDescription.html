MongoDB

  • MongoDB es un sistema de gestión de bases de datos basado en documentos.
  • Open Source.
  • Almacén de datos flexible, potente y escalable.
  • Reemplaza el concepto de "fila" de las bases de datos relacionales por el de "documento".
  • Permite representar relaciones jerárquicas complejas con un solo registro.
  • Reemplaza el concepto de "tabla" de las bases de datos relacionales por el de "colección".
  • Una colección puede almacenar uno o más documentos.
  • Un documento puede tener uno o más campos.
  • Documentos dentro de una misma colección pueden tener una única combinación de campos.
  • Libre de esquema.
  • Sin un esquema, se proporciona gran flexibilidad a la hora de trabajar con modelos de datos que evolucionan continuamente.
