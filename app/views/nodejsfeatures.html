Node.js

  • Plataforma construida sobre el motor de JavaScript de Chrome (V8) para construir fácilmente aplicaciones de red rápidas y escalables.
  • Permite ejecutar JavaScript fuera del navegador (servidores web).
  • Modelo dirigido a eventos y de entrada/salida (I/O) no bloqueante.
  • Ideal para aplicaciones que trabajan de forma intensiva con datos en tiempo real y que corren en dispositivos distribuidos.
  • APIs no bloqueantes optimizadas para trabajar de forma orientada a eventos (manipulación de datos binarios, escritura/lectura en disco...).
  • Simplifica el acceso a los recursos.
  • Elude el reto de la concurrencia sin sacrificar el rendimiento.
  • V8 usa algunas de las nuevas técnicas en tecnología de compilación, lo que mejora el rendimiento.
